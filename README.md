## 📌 Project Overview

The Smart Analytics Tool is an interactive data analysis application developed using Python, Streamlit, Pandas, and Plotly. The purpose of this project is to help users quickly explore and understand datasets without writing complex code. Users can upload any CSV file through the application, and the tool automatically performs essential data analysis tasks.

The application provides a dataset preview, allowing users to inspect the uploaded data. It also displays important dataset information such as the number of rows, columns, and available features. To improve data quality assessment, the tool performs missing value analysis and highlights columns that contain null values.

In addition, the Smart Analytics Tool generates a statistical summary of numerical columns, including measures such as count, mean, standard deviation, minimum, and maximum values. The application also offers dynamic and interactive visualizations, including histograms, scatter plots, and box plots. Users can select different columns to customize the visual analysis according to their needs.

This project demonstrates the practical use of data analysis and visualization techniques in a user-friendly environment. It is designed for students, beginners, and analysts who want a simple yet effective way to explore datasets and gain insights. The tool reduces manual effort and enables faster decision-making through automated analytics and interactive visualizations.
